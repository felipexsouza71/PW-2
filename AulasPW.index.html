Na primeira forma de incluir CSS no HTML, um código CSS é inserido diretamente no "head" do HTML dentro da tag "style"

Na segunda forma de incluir CSS no HTML, deve-se criar um arquivo .css e vincular ele na parte do "head" do HTML usando a tag com nome"link"

Na terceira forma de incluir CSS no HTML, o CSS é aplicado diretamente na tag HTML, usando o atributo chamado style, afetando apenas aquele elemento.
